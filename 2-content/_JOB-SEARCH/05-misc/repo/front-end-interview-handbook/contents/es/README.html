Manual para entrevistas Front End

¿Qué es esto?

A diferencia de las típicas entrevistas de trabajo para ingenieros de software, las entrevistas para trabajos front-end tienen menor énfasis en algoritmos y tienen más preguntas de conocimientos complejos y expertís en el tema — HTML, CSS, JavaScript, por nombrar algunos.

Si bien existen recursos para ayudar a los desarrolladores front-end a prepararse para las entrevistas, no son tan abundantes como los hay para una entrevista de ingeniero de software. Entre los recursos que existen, probablemente el banco de preguntas más útil sería Preguntas de Entrevista para Desarrolladores Front-End. Desafortunadamente, no pude encontrar muchas respuestas completas y satisfactorias a estas preguntas, es por esto que este es mi intento de responderlas. Al ser un repositorio de código abierto, el proyecto vive con el apoyo de la comunidad a medida que evoluciona la web.

💰 ¿Interesado en empleos de desarrollo front end? Regístrate en Triplebyte para entrevistarte eficientemente con las compañías tecnológicas top! 💰

¿Buscas preparación para entrevistas genéricas?

Puede que te interese el Manual para Entrevistas Tecnológicas que tiene contenido útil sobre entrevistas de desarrollo general, como algoritmos, preguntas de comportamiento humano y una hoja resumen de consejos para entrevistas!

Si estás interesado en cómo son implementadas las estructuras de datos, revisa Lago, una biblioteca de estructuras de datos y algoritmos para JavaScript. Prácticamente sigue siendo un trabajo en progreso, pero tengo la intención de convertirlo en una biblioteca que se pueda usar en producción y también como un recurso de referencia para revisar estructuras de datos y algoritmos.
