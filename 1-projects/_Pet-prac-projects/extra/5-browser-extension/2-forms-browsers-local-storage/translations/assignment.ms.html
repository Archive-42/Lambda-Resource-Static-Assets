Menggunakan API

Arahan

API sangat menyeronokkan untuk dimainkan. Berikut adalah senarai banyak percuma. Pilih dan API, dan buat pelanjutan penyemak imbas yang menyelesaikan masalah. Masalah kecil mungkin kerana tidak mempunyai gambar haiwan peliharaan yang cukup (jadi, cubalah API CEO anjing) atau sesuatu yang lebih besar - bersenang-senang!
