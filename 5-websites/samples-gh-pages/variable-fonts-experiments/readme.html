Web Fundamentals.

This is a collection of experiments inspired by axis-praxis.org and typetools.typenetwork.com/. The goal is to demonstrate the flexibility of Variable Fonts and the potential they hold for designers and developers.

Getting Started

These are standalone HTML files and can be run in a browser.

Working Demo be seen at https://variable-font-experiments.glitch.me/
