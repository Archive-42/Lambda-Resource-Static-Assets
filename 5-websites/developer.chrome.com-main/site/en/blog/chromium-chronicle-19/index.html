When dealing with large codebases, it’s useful to have the code indexed, so you can navigate across symbols. For Googlers, CodeSearch works for Google3 and Chromium, but doesn’t yet support Chrome OS, and isn’t available to non-Googlers. Don’t despair, you can use the CLion IDE to get this functionality, in addition to many more useful features.
