Terrarium 배포하기

Azure Static Web Apps를 사용하여 terrarium을 웹에 배포하거나 게시할 수 있습니다.

  1. 저장소를 포크합니다

  2. 버튼을 누릅니다

  1. 앱을 만드는 마법사로 안내합니다. 앱 상위를 /solution 또는 코드베이스의 상위로 설정했는지 확인하십시오. 이 앱에는 API가 없으므로 추가할 필요가 없습니다. Azure Static Web Apps의 빌드 서비스로 빌드하고 새 URL에 앱을 게시하는 데 도움이 되는 포크 리포지토리에 .github 폴더가 생성됩니다.
