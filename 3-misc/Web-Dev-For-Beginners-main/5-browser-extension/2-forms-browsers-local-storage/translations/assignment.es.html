Instrucciones

Puede ser muy divertido jugar con las API. Aquí hay una lista de muchos gratuitos. Elija una API y cree una extensión de navegador que resuelva un problema. Puede ser un problema tan pequeño el no tener suficientes imágenes de mascotas (por lo tanto, pruebe la API de CEO para perros) o algo más grande: ¡diviértase!
