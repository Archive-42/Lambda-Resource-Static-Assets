Adopteer een API

Instructies

API’s kunnen erg leuk zijn om mee te spelen. Hier is een lijst met vele gratis. Kies een API, en bouw een browserextensie die een probleem oplost. Het kan een zo klein probleem zijn als u niet genoeg foto’s van huisdieren hebt (dus probeer de dog CEO API) of iets groters - veel plezier!
