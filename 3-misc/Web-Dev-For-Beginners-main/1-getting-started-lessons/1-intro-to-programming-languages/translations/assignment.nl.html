De Documenten lezen

Instructies

Er zijn veel tools die een webontwikkelaar nodig heeft en die staan in de MDN-documentatie voor client-side tooling. Selecteer 3 tools die niet in de les worden behandeld, leg uit waarom een webontwikkelaar deze zou gebruiken en zoek naar een tool die onder deze categorie valt en deel de documentatie ervan. Gebruik niet hetzelfde toolvoorbeeld voor MDN-documenten.
