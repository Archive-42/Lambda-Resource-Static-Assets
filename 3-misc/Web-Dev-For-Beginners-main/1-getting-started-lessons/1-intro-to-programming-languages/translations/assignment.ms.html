Arahan

Terdapat banyak alat yang mungkin diperlukan oleh pembangun web adalah di Dokumentasi MDN untuk perkakas sisi pelanggan. Pilih 3 alat yang tidak diliputi dalam pelajaran, terangkan mengapa pembangun web menggunakannya, dan mencari alat yang terdapat dalam kategori ini dan kongsikan dokumentasinya. Jangan menggunakan contoh alat yang sama pada MDN.

Rubrik

Contoh Mencukupi Usaha Lagi
Terangkan mengapa pembangun web menggunakan alat Terangkan bagaimana dan mengapa pembangun akan menggunakan alat Tidak menyebut bagaimana atau mengapa pembangun menggunakan alat
