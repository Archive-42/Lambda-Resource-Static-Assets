HTML

Minor markup changes in the selectable and expandable tile. Icons from carbon-elements package are now used. Vanilla markup should be migrated to one shown in carbondesignsystem.com site. React and other framework variants should reflect the change automatically.

Things to note:

  • Tile input is no longer a child of the selectable tile, it’s a sibling of it
  • New icons are used in both the selectable and expandable tiles
